# Calculatree (working name)

## Development
1. Install Bun from https://bun.sh/
2. Install dependencies with `bun i`
3. Start the dev server with `bun dev`

To build the application, use `bun run build`

Linting is done with `bun run lint` and type validation is done with `bun run validate`

**Recommended IDE**: VSCode
> It's also recommended that you install the Typescript and ESLint plugins for VSCode
